# Plano Cartesiano Interativo

Aplicativo HTML para estudo de função afim, pontos no plano cartesiano e reta determinada por dois pontos.

## Correção incluída

A equação agora faz o jogo de sinal corretamente no coeficiente linear:

- Antes: `y = -0.44x + -2.79`
- Agora: `y = -0.44x - 2.79`

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie os arquivos `index.html` e `README.md`.
3. Vá em **Settings > Pages**.
4. Em **Branch**, selecione `main` e `/root`.
5. Salve e aguarde o link ser gerado.
